Après envoi, la demande part dans Supabase avec importe = 0. Votre script Python d'import récupère ensuite la réservation et l'intègre dans le dashboard. Les champs utilisés correspondent à ceux attendus par votre script de synchronisation.

Le script attend notamment : nom, telephone, email, depart, arrivee, date_course, heure, nb_passagers, prestation, notes et importe.
